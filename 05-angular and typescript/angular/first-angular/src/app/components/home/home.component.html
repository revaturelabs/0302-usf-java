intro into data binding

Angular supports data binding. this is a mechanism for coordinating parts of a tempate with parts of a class. There are several different kinds of binding.

First: one way data binding

Interpolation is a way that we can achieve one way data binding. The syntax is {{}} {{name + 2+1}}

Event Binding

{{counter}}

Instead of attaching event listeners, we can just bind the event to the method The syntax goes like this: (event)="function_invocation" Event-Binding is the process of linking actions to dom events. Recall that DOM events can be user interactions like mouse movements, key presses or button clicks, as well as browser actions like load and refresh. In Angular, we can bind component functions to these actions using the event binding syntax.

Property Binding

Elements on a page all have predefined properties. For example, an img element has a src property and input Elements have a disabled property.
